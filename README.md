# Dennis Diepolder - Portfolio v3

Modern portfolio website for Dennis Diepolder, Technical Operations & Service Delivery Manager, featuring advanced scroll-based animations and smooth parallax effects.

## Features

- **Next.js 16.1.1** with React 19 and App Router
- **Tailwind CSS v4** for modern styling
- **GSAP ScrollTrigger** for advanced scroll-based animations
- **Lenis** for buttery smooth scrolling
- **Parallax scroll experience** - sections pin while content animates based on scroll position
- **React Icons** for brand-accurate technology icons
- Fully responsive design
- Scroll progress indicator

## Tech Stack

- Next.js
- React 19
- TypeScript
- Tailwind CSS v4
- GSAP (GreenSock Animation Platform)
- Lenis Smooth Scroll
- React Icons

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
npm start
```

## Project Structure

- `/app/page.tsx` - Main portfolio page with all animations
- `/app/layout.tsx` - Root layout with metadata
- `/app/globals.css` - Global styles and Tailwind directives
- `/public/` - Static assets including profile image

## Sections

1. **Hero** - Introduction with name and title
2. **Profile** - Professional summary
3. **Core Competencies** - Key skills with animated tags
4. **Experience** - Professional work history
5. **Tech Stack** - Technologies and tools (20+ items with spiral animations)
6. **Footer** - Contact information and links

## Animation Highlights

- Pinned sections with scroll-driven content reveals
- Items fly in from various directions with rotation effects
- Sequential tech stack items that spiral in one by one
- Smooth transitions powered by Lenis and GSAP
- Real-time scroll progress bar
